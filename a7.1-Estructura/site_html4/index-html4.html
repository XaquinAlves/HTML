Estructura de una página Web

Estructura HTML4

Con HTML4 los bloque que constituyen una Web se crean con la etiqueta div que permite definir la estructura del documebto Web. Utilando el id de las etiquetas div se utilizará CSS para determinar su posición y su aspecto.

Estructura HTML5

HTML5 apuesta por el uso de etiquetas semánticas que reflejan el contenido de un documento Web para, entro otras funciones, mejorar la clasificación del mismo en buscadores.
